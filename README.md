# Vortex-Verdict
# 漩渦中的力量，驅動最真實的遊戲評價。
# 在漩渦中找到真相，讓我們的評價幫助你做出最好的選擇。
## 要求功能：

- **JS 框架**：Vue.js
- **CSS 框架**：Bootstrap
- **搜尋框**：搜尋功能
- **按鈕動態更改圖片 filter**：搜尋分類
- **header**
- **body**
- **footer**
- **RWD**（響應式設計）

- **可滑動平行欄**
